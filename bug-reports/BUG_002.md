# Bug Report: State Field Shows Concatenated Value

**Bug ID:** BUG_002
**Related Test Case:** TC_PROFILE_001
**Module:** Update Contact Info
**Severity:** Low
**Priority:** Medium

## Summary
After updating profile contact info, the State field displays "CATamilNadu" instead of a single clean state value, suggesting two different inputs were concatenated.

## Steps to Reproduce
1. Log in to ParaBank
2. Click "Update Contact Info"
3. Change the State field value
4. Click Update
5. View "Update Contact Info" page again

## Expected Result
State field should show a single, correctly formatted state value.

## Actual Result
State field displays "CATamilNadu" - a concatenation of two different values.

## Environment
- Application: ParaBank (parabank.parasoft.com)
- Browser: Google Chrome
- Date Found: 2026-07-13

## Notes
Possibly a pre-existing data issue in the demo environment rather than a new defect - worth re-testing with a fresh field entry to confirm reproducibility.
