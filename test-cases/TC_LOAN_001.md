# Test Case: Request Loan

**Test ID:** TC_LOAN_001
**Title:** Verify user can apply for a loan
**Module:** Loans

## Preconditions
User is logged in and has at least one existing account.

## Test Steps
1. Click "Request Loan"
2. Enter loan amount and down payment
3. Select account to apply from
4. Click Apply Now

## Expected Result
Loan is processed and either approved or denied, with a new loan account created if approved.

## Actual Result
Matches expected. Loan approved instantly. Provider: "Wealth Securities Dynamic Loans (WSDL)". New loan account #18561 created.

## Status
Pass
