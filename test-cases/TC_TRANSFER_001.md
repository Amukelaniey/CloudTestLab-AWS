# Test Case: Transfer Funds

**Test ID:** TC_TRANSFER_001
**Title:** Verify user can transfer funds between own accounts
**Module:** Transfers

## Preconditions
User is logged in and has at least two accounts.

## Test Steps
1. Click "Transfer Funds"
2. Enter transfer amount
3. Select "From" account and "To" account
4. Click Transfer

## Expected Result
Transfer completes successfully, confirmation message is shown, and account balances update accordingly.

## Actual Result
Matches expected. $200.00 transferred from account #13344 to #16230. Confirmation: "Transfer Complete!"

## Status
Pass
